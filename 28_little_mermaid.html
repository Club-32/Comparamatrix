Purist Remarks

Diamond Edition Blu-ray is well-received for preserving some grain. Colors closer to older releases, instead of the criticized 2006 version. Most animation changes from the 2006 version have been reversed (restoring original animation errors as well), though the priest's knee is still censored. Original print run had errors, solved through exchange program (what about Europe?)
